# map_dist

地図上で距離を計算・検索できるアプリケーションです。

## 機能

- 地図上で地点を選択し、距離を計算
- 地名検索による地点指定
- シンプルなUI

## ディレクトリ構成

- `app.js` : メインエントリーポイント
- `src/components/` : 画面・UIコンポーネント
- `src/styles/` : スタイル定義
- `src/utils/` : ユーティリティ関数

## 使い方

1. 必要な依存パッケージをインストール
   ```
   npm install
   ```
2. アプリを起動
   ```
   npm start
   ```
